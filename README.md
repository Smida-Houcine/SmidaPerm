# SmidaPerm
EViews add-in for generating permutations of a set of elements using Smida's algorithm.  
Repository for the EViews Add-in: **SmidaPerm**.  
**Description:**   
This repository provides **SmidaPerm**, an EViews Add-in that implements Smida’s algorithm for permutation generation. The Add-in is provided as a compressed **.aipz** file for direct installation within EViews. It is also downloadable from: http://www.eviews.com/Addins/smidaperm.aipz      
**Installation:**   
To install the Add-in, simply run the **.aipz** installer and follow the on-screen instructions within EViews. The provided installer automates both the installation and internal registration of the Add-in within the EViews environment.    
**Usage:**   
Once installed, the Add-in **appears** in the EViews Add-ins menu as “**Smida's Permutation**”.     
**Files:**  
The Add-in is composed by the files:   
• SmidaPerm.prg: Source code.   
• SmidaPerm.install.prg: Installer program.   
• SmidaPerm.pdf: Help file that explains the Add-in.   
• LICENSE.txt: License file.   
• TOC.ini: Configuration file.    
**Requirements:**  
• EViews version: Implemented in EViews 13 and tested in EViews 10.   
• Operating System: Windows 10/11.
